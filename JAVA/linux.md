## 基本操作

`ls` 即list   

- `-a` 即all，查看全部文件，包括隐藏文件  

- `-l` 具体的文件属性

`cd` change directory  可直接加相对路径，加绝对路径时以`/`开头

`pwd` （英文全拼：print work directory） 命令用于显示工作目录。

`mkdir` 创建目录

`rmdir` 删除目录

`cp`  复制文件，详细信息看`cp --help`

`rm -rf 文件名`  强制删除文件

检查版本 

```bash
[root@pychego bin]# cat /etc/redhat-release 
CentOS Linux release 7.9.2009 (Core)
```
