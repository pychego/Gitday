在shell特定文件夹下面输入该命令显示隐藏文件

```
defaults write com.apple.finder AppleShowAllFiles TRUE

killall Finder
```

## pycharm快捷键

F1: 查看文档

F2: 光标跳转到错误处

shift + enter: 直接跳转下一行

option + enter：查看完整的修正列表

control + T: 重构光标处的变量

option + command + L: 格式化代码

command + E: 调出最近文件菜单

control + G :可以同时选中多个同名变量进行编辑

**command + D: 直接复制粘贴光标所在行**

command + delete: 直接删除光标所在行

shift + esc: 隐藏工具窗口
